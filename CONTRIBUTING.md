# 開発貢献ガイド

このプロジェクトへの貢献に興味をお持ちいただきありがとうございます！

## バグ報告・機能リクエスト

バグの報告や新機能の提案は、GitHubのIssueを使用してお願いします。

## プルリクエスト

1. フォークしてブランチを作成
2. コードを変更
3. テストして問題がないことを確認
4. プルリクエストを送信

## コーディング規約

- 適切な命名規則を使用する
- コードにコメントを追加する
- ESLintのルールに従う

## テスト

変更を加える前に、既存の機能が正常に動作することを確認してください。

## ライセンス

プロジェクトへの貢献はMITライセンスの下で提供されます。
